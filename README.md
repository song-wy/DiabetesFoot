# 糖尿病足患者自我管理辅助系统

## 项目简介
基于ASP.NET MVC开发的糖尿病足患者自我管理辅助系统，帮助患者进行血糖监测、伤口护理、药物管理等。

## 功能特性
- 患者信息管理
- 血糖记录与监测
- 伤口护理与照片上传
- 药物管理与提醒
- 健康数据可视化
- 医患交互功能

## 技术栈
- ASP.NET MVC 5
- Entity Framework 6
- SQL Server
- Bootstrap
- Chart.js

## 安装部署
1. 克隆项目
2. 还原NuGet包
3. 配置数据库连接字符串
4. 运行数据库迁移
5. 启动项目

## 开发环境
- Visual Studio 2019/2022
- .NET Framework 4.7.2+
- SQL Server 2012+
